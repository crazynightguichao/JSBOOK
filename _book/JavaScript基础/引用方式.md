# 第四节 引用方式
1. 外部式
```
	<script src="js/index.js"></script>
```
2. 嵌入式
```
	<script></script>
```
3. 超链接

<a href="javascript:alert(我是超链接);">请点击</a>
```
	<a href="javascript:alert(我是超链接);">请点击</a>
``` 
4. 重定向
```
	<form action="javascript:alert(我是重定向);"></form>
```
5. 元素内部事件调用
```
	<div onclick="alert(我是事件调用)">
```
